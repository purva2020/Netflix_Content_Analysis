# Netflix_Content_Analysis
Data Science Project involving Exploratory data analysis(EDA),data cleaning for analysis,Prediction Modelling with a Decision tree ML model and Visualization of data using Python,Pandas,Matplotlib,Seaborn and Scikit-Learn to identify content trends and patterns.

# Customer Purchase Prediction using Decision Tree

## Project Overview
This project uses a Decision Tree Machine Learning model to predict customer purchase behaviour.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Project Steps
- Data Cleaning
- Exploratory Data Analysis
- Model Training using Decision Tree
- Prediction and Evaluation

## Outcome
Built a Decision Tree model to make predictions based on the dataset.
